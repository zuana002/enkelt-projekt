Laste opp webapp på GitHub

1- Jeg skal bruke VS Code til å utvikle en enkelt web app med tittel "PC GitHub Server"

2- I Web app skal jeg utvikle en webside gitt av læreren min (3 poeng)

3- Webside skal få stil ved hjelp av style.css (2 poeng)

4 Etter at jeg er ferdig med det å ha utviklet web app skal jeg laste den opp på GitHub Server

5- For å laste den opp på GitHub Server skal jeg bruke GitHub Desktop (2 poeng)

6- Til slutt skal jeg sende lenke til GitHub kontoen min på Teams i gruppe 1IMA
